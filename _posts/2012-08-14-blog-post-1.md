# ERP Pipeline: Extracting the N2 Component

In this blog post, I share my EEG/ERP analysis pipeline for extracting the N2 component. The example dataset used here comes from one of my previously published studies.

- [Download the dataset](#)
- [Read the published article](#)
- [Download all analysis scripts](#)

## Study Overview

In this study, we investigated whether inhibitory control is involved in voluntary language switching by applying transcranial direct current stimulation (tDCS) over the right dorsolateral prefrontal cortex (rDLPFC), a brain region associated with inhibitory control.

The experiment included three within-subject factors:

- **tDCS Session**: A-tDCS, C-tDCS, S-tDCS  
- **Language**: L1, L2  
- **Language Sequence**: repeat, switch  

The goal of this pipeline is to extract and analyze the N2 ERP component from the EEG recordings.

# ERP Processing Pipeline

The complete ERP pipeline consists of six main steps.

<details>
<summary><strong>Step 1. Load EEG recordings and save merged datasetsas as.set EEGLAB file</strong></summary>

Load raw EEG recordings into EEGLAB and save the merged dataset as a `.set` file for further preprocessing.

</details>

<details>
<summary><strong>Step 2. Preprocess EEG data before ICA</strong></summary>

Perform standard preprocessing procedures, including filtering, re-referencing, channel inspection, and artifact preparation before running ICA.

</details>

<details>
<summary><strong>Step 3. Run Independent Component Analysis (ICA)</strong></summary>

Run ICA to separate neural activity from noise components such as eye blinks, muscle artifacts, and line noise.

</details>

<details>
<summary><strong>Step 4. Remove noisy components and Interpolate bad channels</strong></summary>

Identify and exclude artifact-related ICA components and noisy channels to improve signal quality.

</details>

<details>
<summary><strong>Step 5. Epoch data based on event markers</strong></summary>

Segment continuous EEG data into epochs based on event-locked markers corresponding to the experimental conditions.

</details>

<details>
<summary><strong>Step 6. Extract the N2 component and generate figures</strong></summary>

Compute ERP averages, extract the N2 component, and generate visualization figures for statistical analysis and interpretation.

</details>
