# ERP Pipeline: Extracting the N2 Component

In this blog post, I share my EEG/ERP analysis pipeline for extracting the N2 component. The example dataset used here comes from one of my previously published studies.

- [Download the dataset](#)
- [Read the published article](#)
- [Download all analysis scripts](#)

## Study Overview

In this study, we investigated whether inhibitory control is involved in voluntary language switching by applying transcranial direct current stimulation (tDCS) over the right dorsolateral prefrontal cortex (rDLPFC), a brain region associated with inhibitory control.

The experiment included three within-subject factors:

- **tDCS Session**: A-tDCS, C-tDCS, S-tDCS  
- **Language**: L1, L2  
- **Language Sequence**: repeat, switch  

The goal of this pipeline is to extract and analyze the N2 ERP component from the EEG recordings.

# ERP Processing Pipeline

The complete ERP pipeline consists of six main steps.

<details>
<summary><strong>Step 1. Load EEG Recordings and Merge Datasets</strong></summary>

<p>
The original EEG recordings were collected using a 64-channel EEG system from eegmagine (ANT Neuro) and stored in BrainVision <code>.vhdr</code> format.
</p>

<p>
Each participant completed multiple recording blocks during the experiment.
</p>

<p>
The purpose of this step is to import the raw EEG recordings into EEGLAB, merge all recording blocks into a single continuous dataset, and prepare the data for preprocessing and ERP analysis.
</p>

<hr>

<h2>Initialize EEGLAB Environment</h2>

<p>
The script first initializes the EEGLAB environment and loads the BrainVision import plugin (<code>BVA-io</code>).
</p>

<pre><code>
clc;
clear

eeglab_root = 'D:\setup\matlab\toolbox\eeglab11_0_5_4b';
addpath(genpath(eeglab_root));

plugin_askinstall('BVA-io', [], 1);
</code></pre>

<hr>

<h2>Define Input and Output Directories</h2>

<p>
The input and output directories are defined for loading raw EEG recordings and saving merged datasets.
</p>

<pre><code>
root_dir='D:\0EEG\0_raw_data';
out_dir='D:\0EEG\1_append';
</code></pre>

<hr>

<h2>Loop Through Experimental Conditions</h2>

<p>
The script loops through the three stimulation conditions:
A-tDCS, C-tDCS, and S-tDCS.
</p>

<pre><code>
stimulation={'A-tDCS' 'C-tDCS' 'S-tDCS'};
</code></pre>

<hr>

<h2>Load EEG Recordings</h2>

<p>
For each participant, all BrainVision EEG recordings (<code>.vhdr</code>) are identified and loaded into EEGLAB.
</p>

<pre><code>
data_list=dir('*.vhdr');

EEG = pop_loadbv(perdon_out_dir,data_list(k).name);

EEG = eeg_checkset( EEG );
</code></pre>

<hr>

<h2>Merge Recording Blocks</h2>

<p>
The individual recording blocks are merged into one continuous EEG dataset.
</p>

<pre><code>
EEG = pop_mergeset( ALLEEG, [1 2 3 4 5 6], 0);
</code></pre>

<p>
Merging datasets simplifies preprocessing and creates one participant-level dataset for ERP analysis.
</p>

<hr>

<h2>Save Merged Dataset</h2>

<p>
The merged dataset is saved as an EEGLAB <code>.set</code> file for later preprocessing and ERP analysis.
</p>

<pre><code>
EEG = pop_saveset(EEG,[person_list(i).name,'.set'],Sti_out_dir);
</code></pre>




<hr>

<p>
At the end of this step:
</p>

<ul>
<li>all EEG recordings have been imported into EEGLAB</li>
<li>multiple recording blocks have been merged</li>
<li>participant-level <code>.set</code> files have been generated</li>
<li>the datasets are ready for preprocessing and ICA analysis</li>
</ul>

</p>

<details>
<summary><strong>Step 2. EEG Preprocessing Before ICA</strong></summary>

<p>
The purpose of this step is to preprocess the merged EEG datasets before running Independent Component Analysis (ICA).
</p>

<p>
Several preprocessing procedures are applied to improve data quality, including channel locations, resampling, high pass(0.1) and low pass(30) filtering, re-referencing, exclude bad channels and epoch. 
</p>

<hr>

<h2>Initialize EEGLAB Environment</h2>

<p>
The script first initializes the EEGLAB environment and defines the input and output directories.
</p>

<pre><code>
clc;
clear;

eeglab_root = 'D:\setup\matlab\toolbox\eeglab11_0_5_4b';
addpath(genpath(eeglab_root));

root_dir='D:\0EEG\1_append';
out_dir='D:\0EEG\2_before_ICA';
</code></pre>

<hr>

<h2>Load EEG Datasets</h2>

<p>
The script loops through all stimulation conditions and loads each participant’s merged EEGLAB <code>.set</code> dataset.
</p>

<pre><code>
EEG = pop_loadset(data_list(j).name);
</code></pre>

<hr>

<h2>Remove Additional Channels</h2>

<p>
Additional bipolar channels (<code>BIP</code>) that are not required for ERP analysis are removed from the dataset.
</p>

<pre><code>
EEG = pop_select( EEG,'nochannel',{...});
</code></pre>

<hr>

<h2>Channel Location Editing</h2>

<p>
Standard electrode locations are added using the standard BESA electrode template. The CPz channel label is also corrected.
</p>

<pre><code>
EEG=pop_chanedit(EEG, 'lookup', ... );
</code></pre>

<hr>


<h2>Resampling</h2>

<p>
The EEG data are resampled to 500 Hz to reduce file size and computational load while preserving ERP signals.
</p>

<pre><code>
EEG = pop_resample( EEG, 500);
</code></pre>

<hr>

<h2>Filtering</h2>

<p>
A high-pass filter at 0.1 Hz and a low-pass filter at 30 Hz are applied to remove slow drifts and high-frequency noise.
</p>

<pre><code>
EEG = pop_eegfiltnew(EEG, ...
    'locutoff', 0.1, ...
    'hicutoff',30);
</code></pre>

<hr>

<h2>Re-referencing</h2>

<p>
The EEG signals are re-referenced to the average of the left and right mastoid electrodes (M1 and M2).
</p>

<pre><code>
EEG = pop_reref( EEG, [13 19] );
</code></pre>

<hr>

<h2>Bad Channel Detection</h2>

<p>
Bad channels are automatically identified using variance-based z-scores. Channels with abnormal variance are temporarily excluded before ICA.
</p>

<pre><code>
badchans = find(abs(zscore(var(EEG.data ...
    (chansWtEyes,:,:),[],2)))>3);
</code></pre>

<p>
The removed channel information was saved for later interpolation after ICA. The original data, including the removed channel, was also saved for later analysis.
</p>

<hr>

<h2>Epoching and Artifact Rejection</h2>

<p>
Continuous EEG data are segmented into epochs based on stimulus markers (<code>s4</code>, <code>s5</code>, <code>s6</code>, <code>s7</code>).
</p>

<pre><code>
EEG = pop_epoch( EEG, ...
    { 's4' 's5' 's6' 's7' }, ...
    [-1 2]);
</code></pre>

<p>
Automatic artifact rejection is then applied to remove epochs with excessive amplitudes.
</p>

<pre><code>
[EEG, rmepochs] = pop_autorej( ...
    EEG, ...
    'threshold',150);
</code></pre>

<hr>

<h2>Save Preprocessed Dataset</h2>

<p>
Finally, the preprocessed EEG datasets are saved as EEGLAB <code>.set</code> files for ICA analysis.
</p>

<pre><code>
EEG = pop_saveset( EEG, ...
    'filename', ...
    [data_list(j).name(1:5), ...
    '_chanedit_interp_resemple_filter_ref_epoch.set']);
</code></pre>

<hr>

<p>
At the end of this step:
</p>

<ul>
<li>additional channels have been removed</li>
<li>channel locations have been corrected</li>
<li>data have been filtered and re-referenced</li>
<li>bad channels have been identified</li>
<li>continuous EEG data have been epoched</li>
<li>datasets are ready for ICA decomposition</li>
</ul>
