# ERP Pipeline: Extracting the N2 Component

In this blog post, I share my EEG/ERP analysis pipeline for extracting the N2 component. The example dataset used here comes from one of my previously published studies.

- [Download the dataset](#)
- [Read the published article](#)
- [Download all analysis scripts](#)

## Study Overview

In this study, we investigated whether inhibitory control is involved in voluntary language switching by applying transcranial direct current stimulation (tDCS) over the right dorsolateral prefrontal cortex (rDLPFC), a brain region associated with inhibitory control.

The experiment included three within-subject factors:

- **tDCS Session**: A-tDCS, C-tDCS, S-tDCS  
- **Language**: L1, L2  
- **Language Sequence**: repeat, switch  

The goal of this pipeline is to extract and analyze the N2 ERP component from the EEG recordings.

---

# ERP Processing Pipeline

The complete ERP pipeline consists of six main steps.

<details open>
<summary><strong>Step 1. Load EEG Recordings and Merge Datasets</strong></summary>

The purpose of this step is to import raw EEG recordings into EEGLAB and organize the data into participant-level datasets for later preprocessing and ERP analysis.

The original EEG recordings were collected using  a set of 64 electrodes from eegmagine (ANT Neuro) and stored in `.vhdr` format. Since each participant completed multiple recording blocks during the experiment, the datasets first need to be loaded separately and then merged into a single continuous EEG dataset.

This step prepares the data structure for all subsequent analyses.

---
## 1. Initialize EEGLAB Environment

This section initializes the EEGLAB environment and installs the BrainVision import plugin (`BVA-io`) if it is not already available.

The `BVA-io` plugin allows EEGLAB to read BrainVision EEG files (`.vhdr`, `.vmrk`, and `.eeg`).

```matlab
eeglab_root = 'D:\setup\matlab\toolbox\eeglab11_0_5_4b';
addpath(genpath(eeglab_root));

plugin_askinstall('BVA-io', [], 1);
```

<br>

## 2. Define Input and Output Directories

- `root_dir` contains the original raw EEG recordings.
- `out_dir` is used to save the merged EEGLAB datasets.

The datasets are organized according to stimulation conditions:

- A-tDCS
- C-tDCS
- S-tDCS

```matlab
root_dir='D:\0EEG\0_raw_data';
out_dir='D:\0EEG\1_append';
```

<br>

## 3. Loop Through Experimental Conditions

The script iterates through all stimulation sessions and processes participants separately within each condition.

This structure helps maintain a clear organization of EEG datasets across experimental conditions.

```matlab
stimulation={'A-tDCS' 'C-tDCS' 'S-tDCS'};
```

<br>

## 4. Load EEG Recordings for Each Participant

For each participant, all BrainVision EEG recordings are identified automatically.

Each `.vhdr` file corresponds to one recording block from the experiment.

The recordings are then loaded into EEGLAB using:

```matlab
EEG = pop_loadbv(perdon_out_dir,data_list(k).name);
```

After loading, EEGLAB checks dataset consistency using:

```matlab
EEG = eeg_checkset( EEG );
```

The datasets are temporarily stored in the `ALLEEG` structure for merging.

```matlab
data_list=dir('*.vhdr');
```

<br>

## 5. Merge Multiple Recording Blocks

Because participants completed multiple experimental blocks, the EEG recordings must be merged into a single continuous dataset.

Merging datasets provides several advantages:

- simplifies preprocessing
- allows ICA to be computed on longer continuous data
- ensures consistent preprocessing across blocks
- improves data management

At this stage, no preprocessing or artifact correction has been applied yet.

```matlab
EEG = pop_mergeset( ALLEEG, [1 2 3 4 5 6], 0);
```

<br>

## 6. Save Merged Dataset

The merged EEG dataset is saved as an EEGLAB `.set` file.

The `.set` format preserves:

- EEG signals
- channel information
- event markers
- metadata

These files will be used in the next preprocessing step.

```matlab
EEG = pop_saveset(EEG,[person_list(i).name,'.set'],Sti_out_dir);
```

<br>

## Complete MATLAB Script

```matlab
clc;
clear

eeglab_root = 'D:\setup\matlab\toolbox\eeglab11_0_5_4b';
addpath(genpath(eeglab_root));

plugin_askinstall('BVA-io', [], 1);

root_dir='D:\0EEG\0_raw_data';
out_dir='D:\0EEG\1_append';

cd(root_dir);

stimulation={'A-tDCS' 'C-tDCS' 'S-tDCS'};

for s=1:length(stimulation)

    cd(root_dir);

    Sti_root_dir= fullfile(root_dir, stimulation{s});
    cd(Sti_root_dir)

    person_list=dir('0*');

    for i=1:length(person_list)

        perdon_out_dir=fullfile(Sti_root_dir,person_list(i).name);
        cd(perdon_out_dir)

        data_list=dir('*.vhdr');

        [ALLEEG EEG CURRENTSET ALLCOM] = eeglab;

        for k=1:length(data_list)

            EEG = pop_loadbv(perdon_out_dir,data_list(k).name);

            EEG = eeg_checkset( EEG );

            [ALLEEG EEG CURRENTSET] = pop_newset( ...
                ALLEEG, EEG, 1,'gui','off');

        end

        EEG = pop_mergeset( ALLEEG, [1 2 3 4 5 6], 0);

        Sti_out_dir= fullfile(out_dir, stimulation{s});

        if~exist(Sti_out_dir,'file')
            mkdir(Sti_out_dir)
        end

        EEG = pop_saveset( ...
            EEG, ...
            [person_list(i).name,'.set'], ...
            Sti_out_dir);

    end
end
```
